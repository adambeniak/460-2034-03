# 460-2034/03 – Computer Architectures and Parallel Systems (APPS)

## Course Objectives Expressed as Achieved Skills and Competencies
The course aims to introduce students to the fundamental hardware components of computers and the operational principles of their individual parts. Selected concepts are demonstrated using microcomputers, personal computers, and GPUs (available architectures).
By completing the course, students will gain knowledge of core computer components and understand the general principles of their functionality. During the practical part of the course, students will acquire the ability to develop programs for microcomputers, multiprocessor systems, and GPU programming. They will also become familiar with the basics of assembly language programming.

## Course Annotation
Students will gain knowledge of core computer components and understand the general principles of their functionality.

## Forms of Assessment and Additional Student Requirements
Practical knowledge is verified during laboratory exercises, where individual tasks are evaluated using a points system.
Theoretical knowledge is verified during the final exam.

## Additional Student Requirements
Programming in C, basic knowledge of C++ programming.

## Course Syllabus

### Lectures:
1. Von Neumann and Harvard computer architectures, fundamental characteristics, and principles of operation.
2. Monolithic computers (microcontrollers), design requirements, properties, use cases, and typical integrated peripherals.
3. Fundamental principles of communication with peripherals: buses, I/O ports, program-controlled I/O, and the use of interrupts. The DMA principle.
4. RISC and CISC processors, core features, reasons for creation. Instruction pipelining, branch prediction, types of hazards.
5. The most widespread RISC processors and their properties.
6. Intel processors, history of development, architecture of the latest processors.
7. Memory organization in computers, memory hierarchy. Internal memory (static, dynamic), virtual memory organization.
8. Hard drives and optical discs. Principles of operation, data storage methods, organization of data on the medium.
9. Computer display units: CRT, LCD, OLED, E-Ink.
10. Advanced computer architectures: GPU, history of computing, CUDA technology.
11. Modern trends in computer architectures. Architectures of parallel systems and computers.
12. Machine instructions, basic addressing modes, Intel processor registers in 64-bit mode.
13. Processor Integer Unit, function calls – return values and parameter passing.
14. Stack data organization, local variables, working with floating-point numbers.

### Laboratory Exercises:
1. Safety training, introduction to the microcomputer development kit and the programming environment, programming a simple application.
2. The principle of Pulse Width Modulation (PWM), LED control, simple animation.
3. Pulse Width Modulation (PWM), parallel LED control, RGB color mixing, control using buttons.
4. LCD graphical display control, color mixing, bitmap font processing.
5. LCD control, displaying graphical and textual information, simple button-controlled application.
6. I2C bus, I/O expander and LED control.
7. I2C bus, FM radio module control, displaying RDS information.
8. CUDA technology, fundamental program concept, programming an application using vectors and matrices.
9. CUDA technology, digital image processing, programming basic transformations.
10. CUDA technology, creating simple animations.
11. Basics of Assembly Language (ASM) programming, linking with C language, addressing, use of global variables.
12. Basic ALU instructions, calling ASM functions from C language, parameter passing, local variables.
13. Floating-point parameter passing.
14. Utilization options of the SSEx unit.
